# AI Job Recommendation System

## 📌 Description
This project recommends jobs based on resume using NLP techniques.

## 🚀 Features
- Resume parsing
- Skill extraction
- Job matching
- API + Frontend UI

## 🛠 Tech Stack
- Python
- FastAPI
- NLP (spaCy)
- Streamlit

## ▶️ How to Run
```bash
pip install -r requirements.txt
uvicorn backend.main:app --reload
